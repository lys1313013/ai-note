# A2A协议


A2A：开启Agent间的自然协作

在AI Agent的世界里，主要解决两大互联领域的挑战：

第一、Agent于Tools（工具）的交互
Agent需要调用外部API、访问数据库、执行代码等。

第二、Agent于Agent（其他智能体或用户）的交互
Agent需要理解其他Agent的意图、协同完成任务、与用户进行自然的对话。
