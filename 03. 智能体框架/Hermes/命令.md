# Hermes 命令

Hermes 智能体框架的常用命令。

## 启动

```bash
hermes
```

## 常用选项

- `--config <path>` — 指定配置文件路径
- `--verbose` — 详细日志
- `--port <port>` — 服务端口
- `--reload` — 热重载（开发模式）

## 配置

配置文件默认位置 `~/.hermes/config.yaml`，主要配置项：

```yaml
model:
  provider: openai
  api_key: sk-xxx
  base_url: https://api.openai.com/v1
  model: gpt-4

agent:
  max_iterations: 10
  timeout: 60

tools:
  - name: web_search
  - name: code_execution
```

## 调试

```bash
# 查看日志
hermes logs

# 测试某个工具
hermes test-tool web_search
```
