# 添加新的内置工具


以生成 uuid 举例
核心实现层
新增文件：tools/uuid_generator.py
- 作用：定义工具的业务逻辑、输入参数描述（Schema）并注册到全局 registry
- 机制：启动时 tools/*.py，会被自动扫描和加载，无需修改任何现有 import 语句
逻辑分组层
修改文件 toolsets.py
- 修改内容：在 TOOLSETS 字典中新增 utility 分组。并在`_HERMES_CORE_TOOLS`列表加入`"generate_uuid"`(这使得该工具能被基础配置继承)
- 作用：将零散的工具（tools）打包成逻辑集合（toolset)。Agent 启动时，是按 toolset 来分配能力的。
UI 暴露层
修改文件 hermes_cli/tools_config.py
- 修改内容：在`CONFIGURABLE_TOOLSETS`列表中添加一行("utility", "🔧 Utility Tools", "UUID generation, etc.") 。
- 作用：让用户能在`hermes tools`在中断图形界面看到这个工具集。
