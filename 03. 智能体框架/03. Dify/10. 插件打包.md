

# 一 打在线包

特点：包比较小，导入Dify平台后，Dify还需要联网下载依赖

https://www.zhihu.com/question/15600173332/answer/1888664502335160759



打包工具下载地址：
https://github.com/langgenius/dify-plugin-daemon/releases



选择 `dify-plugin-windows-amd64.exe` 还是 `dify-plugin-windows-arm64.exe`


1. 按下 `Win + R`，输入 `msinfo32`，回车。
2. 在“系统信息”窗口中，查看“系统类型”：

    1. 如果显示 **x64-based PC**，请选择 `dify-plugin-windows-amd64.exe`。
    2. 如果显示 **ARM-based PC**，请选择 `dify-plugin-windows-arm64.exe`




打包命令


```
./dify-plugin-windows-amd64.exe plugin package dify-plugin-tools-mcp_sse/
```
