# Dify 数据库表

Dify 平台核心表结构（PostgreSQL）。

## 核心表

- **conversations** — 对话会话表
  - id, app_id, user_id, created_at
- **messages** — 对话消息表
  - id, conversation_id, role, content, created_at
- **apps** — 应用表
  - id, name, mode (chat/workflow/completion)
- **datasets** — 知识库表
- **documents** — 文档表
- **document_segments** — 分段表

## 表关系

```
apps (1) ── (N) conversations (1) ── (N) messages
  │
  └── (N) datasets (1) ── (N) documents (1) ── (N) document_segments
```

## 常用查询

```sql
-- 查看某应用的所有会话
SELECT * FROM conversations WHERE app_id = 'xxx' ORDER BY created_at DESC;

-- 查看某会话的消息
SELECT * FROM messages WHERE conversation_id = 'xxx' ORDER BY created_at;

-- 知识库文档统计
SELECT dataset_id, COUNT(*) FROM documents GROUP BY dataset_id;
```
