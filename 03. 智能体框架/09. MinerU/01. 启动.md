# 启动

```bash
git clone https://github.com/opendatalab/MinerU.git
```


```bash
 uv pip install -e ".[core]" -i https://mirrors.aliyun.com/pypi/simple
```


```bash
mineru-api --host 0.0.0.0 --port 8000
```

调用参数 (可能已经是旧版本了)
```
curl --location --request POST 'http://127.0.0.2:8888/file_parse' \
--form 'files=@"/Users/hurry/Downloads/invoice.pdf"' \
--form 'output_dir="./output"' \
--form 'lang_list="ch"' \
--form 'parse_method="auto"' \
--form 'return_md="true"' \
--form 'return_images="true"' \
--form 'return_middle_json="false"' \
--form 'start_page_id="0"' \
--form 'end_page_id="5"'
```
