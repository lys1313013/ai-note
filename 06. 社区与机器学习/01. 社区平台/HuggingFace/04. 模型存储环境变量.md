# HuggingFace 模型存储环境变量

控制 HuggingFace 模型和数据集的本地存储位置。

![image.png](04. 模型存储环境变量.assets/20250329153629.png)

## 核心环境变量

| 变量 | 作用 | 默认 |
|------|------|------|
| `HF_HOME` | 总根目录 | `~/.cache/huggingface` |
| `HUGGINGFACE_HUB_CACHE` | 模型/数据集缓存 | `$HF_HOME/hub` |
| `TRANSFORMERS_CACHE` | transformers 缓存（旧版） | `$HF_HOME/hub` |
| `HF_DATASETS_CACHE` | datasets 库缓存 | `$HF_HOME/datasets` |
| `HF_HUB_OFFLINE` | 离线模式（1=仅本地） | 0 |
| `HF_HUB_DISABLE_PROGRESS_BARS` | 关闭进度条 | 0 |
| `HF_TOKEN` | 认证 token | - |

## 修改存储位置

### 临时

```bash
export HF_HOME=/path/to/your/cache
```

### 永久（写入 ~/.zshrc 或 ~/.bashrc）

```bash
echo 'export HF_HOME=/Volumes/External/huggingface' >> ~/.zshrc
source ~/.zshrc
```

## 修改镜像（中国大陆）

```bash
export HF_ENDPOINT=https://hf-mirror.com
```

## Python 使用

```python
import os
os.environ["HF_HOME"] = "/path/to/cache"

from transformers import AutoModel
model = AutoModel.from_pretrained("bert-base-chinese")
```

## 清理缓存

```bash
# 查看大小
du -sh ~/.cache/huggingface

# 删除旧版本
huggingface-cli delete-cache
```
