# 前言

大模型应用本质就是调http 接口。
大模型本身没有任何记忆、工具调用的能力。记忆的能力只是通过智能体把历史的对话再次传给了大模型，工具调用只是罗列了智能体有哪些工具，让大模型选择一个工具，并提供参数。



# 接口

## 补全接口

/chat/completions

|                 | 类型   |      |
| --------------- | ------ | ---- |
| model           | 字符串 |      |
| messages        |        |      |
| stream          |        |      |
| tools           |        |      |
| response_format |        |      |



```json
{
	"model": "",
	
}
```


示例：

```json
{
  "model" : "deepseek-reasoner",
  "messages" : [ {
    "role" : "user",
    "content" : "今天是哪一天？今天的上海的天气怎么样？"
  } ],
  "stream" : false,
  "tools" : [ {
    "type" : "function",
    "function" : {
      "name" : "getTime",
      "description" : "获取当前时间",
      "parameters" : {
        "type" : "object",
        "properties" : { },
        "required" : [ ]
      }
    }
  }, {
    "type" : "function",
    "function" : {
      "name" : "getWeather",
      "description" : "获取某一个具体城市的天气",
      "parameters" : {
        "type" : "object",
        "properties" : {
          "city" : {
            "type" : "string",
            "description" : "指定的城市"
          },
          "time" : {
            "type" : "string",
            "description" : "具体时间"
          }
        },
        "required" : [ "city", "time" ]
      }
    }
  } ]
}
```
