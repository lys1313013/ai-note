# OCR

光学字符识别（Optical Character Recognition）工具与方案。

## 主流工具

- **PaddleOCR**（百度飞桨）— 中文场景首选，支持 80+ 语言
- **Tesseract**（Google）— 老牌开源，多语言支持好
- **EasyOCR** — 简单易用，深度学习驱动
- **PaddleOCR-VL** — 视觉大模型版，多模态理解
- **商业 API** — 百度/腾讯/阿里云 OCR，识别精度高

## PaddleOCR 快速使用

```bash
pip install paddleocr
```

```python
from paddleocr import PaddleOCR
ocr = PaddleOCR(use_angle_cls=True, lang='ch')
result = ocr.ocr('image.png', cls=True)
for line in result[0]:
    print(line[1][0])  # 识别文字
```

## 场景选型

| 场景 | 推荐 |
|------|------|
| 印刷体中文 | PaddleOCR / 商业 API |

| 多语言混合 | PaddleOCR / Tesseract |
