# 语音转文字（ASR）

自动语音识别（Automatic Speech Recognition）工具与方案。

## 主流工具

- **Whisper**（OpenAI）— 开源多语言模型，支持 99 种语言
  - tiny / base / small / medium / large
  - 可本地部署，CPU 也能跑 small
- **FunASR**（阿里达摩院）— 中文优化，工业级
- **Paraformer**（阿里）— 非自回归架构，速度快
- **PaddleSpeech**（百度）— 中文场景

## Whisper 快速使用

```bash
# 安装
pip install openai-whisper

# 命令行
whisper audio.mp3 --language Chinese --model medium

# Python API
import whisper
model = whisper.load_model("base")
result = model.transcribe("audio.mp3", language="zh")
print(result["text"])
```

## 选型建议

| 场景 | 工具 |
|------|------|
| 通用多语言 | Whisper large-v3 |
| 中文为主 | FunASR / Paraformer |
| 实时流式 | Whisper streaming / Vosk |
| 高精度 | 商业 API（讯飞/阿里云） |
