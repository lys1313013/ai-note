# Ollama 模型内容

Ollama 管理的模型目录结构和内容说明。

## 模型存储位置

默认位置：`~/.ollama/models/`

可通过环境变量修改：

```bash
export OLLAMA_MODELS=/path/to/your/models
```

> macOS 推荐放在外部 SSD 上

## 目录结构

```
~/.ollama/models/
├── blobs/              # 模型权重文件（实际的二进制）
│   ├── sha256-xxxx
│   └── sha256-yyyy
└── manifests/          # 模型元数据（manifest）
    └── registry.ollama.ai/
        └── library/
            └── llama3/
                └── latest
```

## 关键文件

- **blobs/** — 存储模型权重文件（如 GGUF 格式）
- **manifests/** — 记录每个模型对应的 blob、参数、模板等元信息

## 模型格式

Ollama 使用 **GGUF**（GPT-Generated Unified Format）格式，由 llama.cpp 项目定义。

## 常用命令

```bash
# 查看已下载模型
ollama list

# 查看模型详情
ollama show llama3

# 删除模型
ollama rm llama3

# 复制模型（用于自定义）
ollama cp llama3 my-llama3
```
