## 五、常见问题解决

**权限问题**

- 如果在 Linux 或 macOS 系统上遇到权限问题，可以尝试运行以下命令：
    
    bash复制
    
    sudo usermod -aG ollama $USER
    
- 然后重新启动 Ollama。
    

**模型存储路径**

- 默认情况下，模型存储路径为 `C:\Users\<用户名>\.ollama`（Windows）或 `~/.ollama`（Linux/Mac）。
    
- 如果需要修改存储路径，可以在启动时添加参数（如 Windows 系统）：
    
    bash复制
    
    --ollama-dir D:\new_path
    
- 或在 Linux 系统中通过环境变量设置。



### （三）Linux 系统安装

##### 一键安装

- 打开终端，运行以下命令进行安装：
    
    curl -fsSL https://ollama.com/install.sh | sh
    
- 安装完成后，运行 `ollama -v` 验证安装是否成功。
    

**手动安装（可选）**

- 如果需要手动安装，可以运行以下命令：
    
    mkdir -p ~/ollama/bin  
    curl -L https://ollama.com/download/ollama-linux-amd64 -o ~/ollama/bin/ollama  
    chmod +x ~/ollama/bin/ollama  
    export PATH=$PATH:$HOME/ollama/bin
    
- 然后运行 `ollama -v` 验证安装。
