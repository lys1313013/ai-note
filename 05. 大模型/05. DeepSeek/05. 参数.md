# 参数

DeepSeek 推理时的关键采样参数。

## 常用参数

- **temperature** — 采样温度，控制输出随机性。值越大越发散（0~2，0 为贪心，1 为平衡）
- **top_p** — 核采样，累积概率阈值（0~1，常用 0.9）
- **top_k** — 仅从概率最高的 k 个 token 中采样
- **max_tokens** — 单次生成的最大 token 数
- **presence_penalty** — 主题重复惩罚
- **frequency_penalty** — 字面重复惩罚
- **stop** — 停止序列
- **stream** — 是否流式返回

## 推荐组合

| 场景 | temperature | top_p |
|------|-------------|-------|
| 代码生成 | 0~0.2 | 0.95 |
| 通用对话 | 0.7 | 0.9 |
| 创意写作 | 0.9~1.2 | 0.95 |
| 事实问答 | 0 | - |

## DeepSeek-R1 推理模型

DeepSeek-R1 类推理模型建议：

- temperature = 0.6
- top_p = 0.95
- 关闭 system prompt 中的"思维过程"指令，让模型自主推理
