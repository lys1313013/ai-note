# LoRA

Low-Rank Adaptation，低秩自适应，一种轻量化微调方法。

## 核心思想

冻结预训练模型权重，在线性层旁路注入两个低秩矩阵 A·B（`A ∈ R^(d×r)`，`B ∈ R^(r×k)`，`r ≪ d,k`），仅训练 A、B。

$$W' = W + \Delta W = W + \frac{\alpha}{r} BA$$

- **r** — 秩，常用 4/8/16
- **α** — 缩放系数，常设为 r 的 2 倍
- **target_modules** — 注入位置（Q/K/V/O/FFN）

## 超参数

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| r | 8/16 | 越大拟合能力越强 |
| alpha | 16/32 | 通常 = 2r |
| dropout | 0.05 | 防过拟合 |
| target_modules | q_proj, v_proj | 全参数注入会接近全量微调 |
| learning_rate | 1e-4 ~ 5e-4 | 比全量微调大 |
| epochs | 3~5 | 数据量小易过拟合 |

## 变体

- **QLoRA** — 4-bit 量化 + LoRA，显存更省
- **AdaLoRA** — 自适应秩分配
- **LoRA+** — A、B 矩阵使用不同学习率
