# 什么是智能体

OpenAI 研究主管 **Lilian Weng** 给出的经典定义：

> **Agent = 大模型（LLM）+ 规划（Planning）+ 记忆（Memory）+ 工具使用（Tool Use）**

## 四大核心组件

| 组件 | 作用 | 典型实现 |
|------|------|----------|
| **LLM** | 推理与决策的大脑 | GPT-4 / Claude / DeepSeek |
| **Planning** | 任务拆解、计划生成 | CoT、ReAct、Plan-and-Execute |
| **Memory** | 上下文与历史 | 短期（Context）+ 长期（向量库） |
| **Tool Use** | 与外部世界交互 | Function Calling、MCP |

## 分类

按自主程度：

- **简单 LLM** — 单次问答，无工具
- **Workflow** — 预定义流程，可调用 LLM（如 Dify 工作流）
- **Agent** — 自主决策、动态调用工具（如 LangGraph Agent）
- **Multi-Agent** — 多个 Agent 协作（如 AutoGen、CrewAI）

## 应用场景

- 代码开发（Devin、Claude Code）
- 深度研究（Deep Research、Gemini Research）
- 数据分析（Agent 写 SQL + 执行）
- 浏览器自动化（Browser Use）
- 办公自动化（自动操作 Office/邮件）
