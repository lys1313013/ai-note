创建我的儿子

```
// 查找名为 '我' 且年龄为 26 的 Person 节点  
MATCH (p:Person {name: '我', age: 26})  
// 创建一个新的 Person 节点  
CREATE (son:Person {name: '儿子的名字', age: 24})  
// 在两个节点之间创建一个 "儿子" 关系  
CREATE (p)-[:我的儿子]->(son)  
RETURN p, son;
```
