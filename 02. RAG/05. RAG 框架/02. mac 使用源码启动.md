# mac 使用源码启动 RagFlow

在 macOS 上从源码启动 RagFlow 完整流程。

## 环境准备

```bash
# 安装 uv（Python 包管理器）
brew install uv

# 克隆仓库
git clone https://github.com/infiniflow/ragflow.git
cd ragflow
```

## 启动后端

```bash
# 安装依赖
uv sync

# 启动开发服务
uv run api/ragflow_server.py
```

## 启动前端

```bash
cd web
npm install
npm run dev
```

## Docker 启动（备选）

如源码启动遇到问题，可直接用 Docker：

```bash
docker compose up -d
```

## 常见问题

- **端口冲突** — RagFlow 默认 9380，前端 9222
- **模型下载慢** — 配置 HuggingFace 镜像 `HF_ENDPOINT=https://hf-mirror.com`
- **内存不足** — 修改 `docker-compose.yml` 调整 ES 内存
