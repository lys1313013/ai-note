# 启动


```
pip install uv
```


下载全部的依赖
```bash
uv pip install -e .[full] --python 3.10
uv run download_deps.py
```

# 授权


```bash
sudo chown -R ubuntu:ubuntu /home/ubuntu/ragflow
```


# 启动基础环境
```bash
docker compose -f docker-compose-base.yml down -v
docker compose -f docker-compose-base.yml up -d
```
