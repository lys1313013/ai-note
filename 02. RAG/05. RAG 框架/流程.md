# RagFlow 流程

RagFlow 是一款开源的 RAG 引擎，专注于**深度文档理解**和**复杂格式问答**。

## 整体流程

```
文档上传 → 解析（Layout Analysis）→ 分块（智能分块）→ Embedding
                                                         ↓
用户查询 → Query 改写 → 向量检索 + 全文检索 → Rerank → LLM 生成
```

## 核心组件

| 组件 | 作用 |
|------|------|
| **DeepDoc** | 文档解析（PDF/Word/Excel/PPT/图片） |
| **Document Parser** | 版面分析、表格识别、公式识别 |
| **Chunking** | 智能分块（按语义/段落/标题） |
| **Embedding** | 向量化（支持多种模型） |
| **Retrieval** | 混合检索（向量 + BM25） |
| **Rerank** | 二阶段重排序 |
| **Generation** | LLM 生成（含引用溯源） |

## 与朴素 RAG 的差异

| 维度 | 朴素 RAG | RagFlow |
|------|----------|---------|
| 文档解析 | 简单切分 | DeepDoc 深度解析 |
| 分块 | 固定长度 | 智能语义分块 |
| 检索 | 单向量 | 向量 + 全文混合 |
| 引用 | 无 | 答案带来源标注 |
| 复杂文档 | 支持差 | PDF 表格/公式友好 |
