# 嵌入模型（Embedding Model）

嵌入（Embedding）指把高维、稀疏、非结构化数据（如文本、图片、音频）映射到**低维、稠密、连续向量空间**中的过程。

> "把东西塞进一个更紧凑、更有意义的空间里"

## 为什么需要嵌入

- 语义可计算 — 向量距离反映语义相似度
- 维度可控 — 文本可以是几千 token，压缩为 768/1024 维
- 支持检索 — 配合向量数据库做相似度搜索

## 工作原理（以文本为例）

```
"今天天气真好" → [0.12, -0.34, 0.56, ..., 0.78]   (768 维)
"阳光明媚的日子" → [0.15, -0.31, 0.58, ..., 0.75]   (相似)
"明天下雨"       → [-0.22, 0.45, -0.12, ..., 0.31]   (不相似)
```

通过大规模对比学习，让语义相近的文本向量距离近。

## 主流嵌入模型

| 模型 | 维度 | 特点 |
|------|------|------|
| **OpenAI text-embedding-3-small** | 1536 | 性价比高 |
| **OpenAI text-embedding-3-large** | 3072 | 精度高 |
| **BGE-M3**（智源） | 1024 | 中文优秀、多语言 |
| **BGE-large-zh-v1.5** | 1024 | 中文专用 |
| **M3E** | 768 | 轻量中文 |
| **Cohere embed-v3** | 1024 | 商业级多语言 |
| **bge-reranker-large** | - | 重排序用 |

## 选型建议

- **中文为主** → BGE 系列、M3E
- **多语言** → BGE-M3、Cohere embed-v3
- **追求质量** → OpenAI text-embedding-3-large
- **本地部署** → BGE / M3E（开源，可商用）

## 使用示例（OpenAI）

```python
from openai import OpenAI
client = OpenAI()

resp = client.embeddings.create(
    model="text-embedding-3-small",
    input="今天天气真好",
)
vector = resp.data[0].embedding  # 1536 维
```

## 使用示例（开源 BGE）

```python
from sentence_transformers import SentenceTransformer
model = SentenceTransformer("BAAI/bge-large-zh-v1.5")
vec = model.encode("今天天气真好")  # 1024 维
```

```
                              ↓

```
