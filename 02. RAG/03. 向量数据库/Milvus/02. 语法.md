# Milvus 过滤表达式语法

Milvus 在向量检索时支持标量字段过滤，常用语法：

## 基础比较

```sql
field == value
field != value
field > value
field < value
field >= value
field <= value
```

## 逻辑运算

```sql
field > 10 AND field < 100
field == "active" OR field == "pending"
NOT (field > 100)
```

## 模糊匹配

```sql
text like "%%keyword%%"        -- 包含
metadata["fi_rule_desc"] like "%%"  -- JSON 字段
text like "prefix_%"           -- 前缀
text like "%_suffix"           -- 后缀
```

## JSON 字段

Milvus 支持 JSON 类型字段，访问子字段用 `[]`：

```sql
metadata["category"] == "tech"
metadata["tags"][0] == "urgent"
```

## IN / NOT IN

```sql
status IN ["active", "pending"]
status NOT IN ["deleted", "archived"]
```

## 示例

```python
results = client.search(
    collection_name="docs",
    data=[query_vector],
    filter='status == "published" AND metadata["category"] == "tech"',
    limit=10,
)
```
