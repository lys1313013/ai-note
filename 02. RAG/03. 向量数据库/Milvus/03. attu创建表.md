# Attu 创建表

使用 Attu（Milvus 官方 GUI）创建 Collection 的步骤。

![image.png](03. attu创建表.assets/20250902105950.png)

## 步骤概览

1. 连接 Milvus 服务（默认 `localhost:19530`）
2. 左侧菜单点击 **Collection** → **Create Collection**
3. 填写：
   - **Collection Name** — 集合名
   - **Primary Field** — 主键（默认 `id`，INT64 或 VARCHAR）
   - **Vector Field** — 向量字段
     - **Field Name** — `vector` 或自定义
     - **Dimension** — 768 / 1024 / 1536 等
     - **Metric Type** — `L2` / `IP` / `COSINE`
4. 添加标量字段（可选）：
   - VARCHAR / INT64 / JSON / BOOL / FLOAT
5. 点击 **Create** 完成

## 索引设置

创建后建议添加索引：

- **HNSW** — 高精度，内存占用大
- **IVF_FLAT** — 平衡性能与精度
- **IVF_SQ8** — 量化压缩，省内存
- **DISKANN** — 大规模、磁盘索引

## 加载集合

创建后需要 `Load` 才能查询：

- Collection → 选择 → **Load**
