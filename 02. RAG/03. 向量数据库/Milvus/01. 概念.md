# Milvus

Milvus 是面向 AI 应用的云原生开源向量数据库。

## 核心特性

- 亿级向量毫秒级检索
- 支持多种索引（HNSW、IVF、ANNOY 等）
- 多种相似度度量（IP、L2、Cosine）
- 标量字段过滤（Hybrid Search）
- 分布式架构，易水平扩展
- 支持 GPU 加速

## 安装（Standalone Docker）

```bash
# 1. 下载 docker-compose 配置
wget https://github.com/milvus-io/milvus/releases/download/v2.4.0/milvus-standalone-docker-compose.yml

# 2. 启动
docker compose -f milvus-standalone-docker-compose.yml up -d

# 3. 查看状态
docker compose -f milvus-standalone-docker-compose.yml ps
```

启动后端口：

- 19530 — gRPC
- 9091 — HTTP
- 9000 — S3 存储（etcd 用）

## Attu 可视化

```bash
docker run -d --name attu -p 8000:3000 zilliz/attu:latest
```

访问 http://localhost:8000，连接 `milvus-standalone:19530`

## 核心概念

- **Collection** — 类似关系库的 Table
- **Entity** — 一行记录
- **Field** — 列（标量字段 + 向量字段）
- **Index** — 向量字段的索引
- **Partition** — 集合内分区，提升查询效率

## Python SDK

```python
from pymilvus import MilvusClient

client = MilvusClient(uri="http://localhost:19530")

# 创建集合
client.create_collection(
    collection_name="my_docs",
    dimension=768,  # 向量维度
    metric_type="COSINE",
)

# 插入
data = [
    {"id": 1, "vector": [...], "text": "hello"},
    {"id": 2, "vector": [...], "text": "world"},
]
client.insert(collection_name="my_docs", data=data)

# 检索
results = client.search(
    collection_name="my_docs",
    data=[[0.1, 0.2, ...]],  # 查询向量
    limit=5,
    output_fields=["text"],
)
```
